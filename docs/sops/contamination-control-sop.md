# Contamination Control SOP

**Mandatory contamination prevention for all film and coating work**

---

## Purpose

Prevent fisheyes, adhesion failure, and rework caused by environmental and handling contamination.

## Rules — non-negotiable

<ul class="checklist">
<li>No silicone, tire shine, or aerosol products in install bays</li>
<li>No cardboard on install tables — use clean work surfaces only</li>
<li>Wash hands before handling film; no hand lotion in bays</li>
<li>Change tack cloths and blades per panel or when contaminated</li>
<li>Close doors during active install — control airflow and dust</li>
<li>Pets, food, and drinks prohibited in production bays</li>
</ul>

## Daily verification

Use **Daily Bay Reset Checklist** at open and between jobs when switching service types.
