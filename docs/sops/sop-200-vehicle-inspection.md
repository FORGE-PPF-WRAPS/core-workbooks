# SOP-200 — Vehicle Inspection

**Forge Standard Operating Procedures v1.0**

**Section 3: Installation Operations**

---

## Purpose

Assess paint, glass, and substrate condition before any installation work begins.

## Scope

Lead installer or designated prep tech. After CS check-in, before wash or decon.


## Procedure

<div class="procedure">

### Lighting & documentation

1. Position vehicle in inspection lighting — minimum 1000 lux on horizontal surfaces.
2. Verify CS inspection form on file; re-walk if gaps exist.
3. Photograph every panel at 3 angles minimum.

</div>

<div class="procedure">

### Paint & substrate assessment

1. Note swirls, chips, rust, repaints, PDR, and aftermarket parts.
2. Identify high-risk areas: rubber trim, badges, sensors, panoramic roofs.
3. Flag paint correction requirement before film install — stop if customer declined correction on swirled panels.

</div>

<div class="procedure">

### Production notes

1. Enter findings on work order — installer-facing notes.
2. Complete Pre-Install QC Checklist.
3. Escalate scope changes to manager before proceeding.

</div>

## Quality standards

<ul class="checklist">
<li>Pre-install checklist on 100% of jobs</li>
<li>No film applied over undisclosed pre-existing damage</li>
</ul>
