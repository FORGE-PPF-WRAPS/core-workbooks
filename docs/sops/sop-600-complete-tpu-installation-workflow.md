# SOP-600 — Complete TPU Installation Workflow

**Forge Standard Operating Procedures v1.0**

**Section 7: Colored TPU**

---

## Purpose

End-to-end procedure for colored TPU (paint protection with color) installation.

## Scope

Colored TPU color-change and protection packages. Combines PPF technique with wrap-level coverage planning.


## Procedure

<div class="procedure">

### Pre-install

1. Complete SOP-200 through SOP-204.
2. Verify color batch match across all rolls — same lot number.
3. Plan seam placement away from primary sight lines.

</div>

<div class="procedure">

### Surface prep

1. Same standard as SOP-400 — wash, decon, IPA per panel.
2. Higher paint correction standard — TPU shows defects more than clear PPF.

</div>

<div class="procedure">

### Application

1. Handle TPU with gloves — oils affect color coat.
2. Slip solution ratio per TPU manufacturer — often less slip than clear PPF.
3. Squeegee with felt edge; avoid repeated passes on color face.
4. Post-heat to spec — overheating yellows or hazes color TPU.

</div>

<div class="procedure">

### Seams & edges

1. Overlap seams in hidden areas; use manufacturer-approved seam tape if required.
2. Edge seal with primer only where manual specifies.

</div>

<div class="procedure">

### QC

1. Color consistency check under multiple light sources.
2. Complete SOP-205 and SOP-206.

</div>

## Quality standards

<ul class="checklist">
<li>Batch lot recorded on work order</li>
<li>No color haze or orange peel from over-heating</li>
</ul>
