# SOP-203 — Installation Bay Setup

**Forge Standard Operating Procedures v1.0**

**Section 3: Installation Operations**

---

## Purpose

Configure a clean, organized bay for maximum quality and efficiency.

## Scope

Before every job. Lead installer or assigned tech.


## Procedure

<div class="procedure">

### Bay environment

1. Complete Daily Bay Reset Checklist.
2. Verify lighting — no burned-out LEDs in primary work zone.
3. Dust sweep; damp-mop if floor visible dirty.
4. Close bay doors during active install to limit dust ingress.

</div>

<div class="procedure">

### Equipment layout

1. Heat gun on stand; extension cords routed overhead, not across walk paths.
2. Solution bottles labeled and filled; blades in dispenser.
3. Plotter warmed up if computer-cut job.
4. Trash and film liner bins within arm's reach.

</div>

<div class="procedure">

### Job staging

1. Pull work order; confirm film lot numbers match job card.
2. Stage patterns or roll film at bay entry.
3. Post "Bay Occupied" sign if customers have bay sightlines.

</div>

## Quality standards

<ul class="checklist">
<li>Bay reset between every job</li>
<li>Film lot traceable to work order</li>
</ul>
