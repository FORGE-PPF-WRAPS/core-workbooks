# SOP-700 — Creating Invoices

**Forge Standard Operating Procedures v1.0**

**Section 8: Business Operations**

---

## Purpose

Generate accurate invoices that match authorized scope and support accounting.

## Scope

Customer service or manager at job completion.


## Procedure

<div class="procedure">

### Invoice build

1. Pull work order — verify scope matches signed estimate or change orders.
2. Line-item labor, materials, shop supplies, tax, and discounts.
3. Apply deposit credit; calculate balance due.

</div>

<div class="procedure">

### Review

1. Manager approval if variance from estimate exceeds 10%.
2. Attach change order signatures if scope changed.
3. Generate PDF invoice; store in CRM.

</div>

## Quality standards

<ul class="checklist">
<li>Invoice issued same day as delivery</li>
<li>Zero invoices without linked work order</li>
</ul>
