# Marketing Resources Kit

**Templates, standards, and campaigns for shop growth**

---

## Purpose

Operational marketing resources that complement the Customer Service Operations Manual (Chapters 14, 24–26). Use for social media, referrals, service one-pagers, and email campaigns.

---

## Service one-pagers

Create one PDF per service using branded theme. Each one-pager includes:

| Section | Content |
|---------|---------|
| Headline | Primary benefit (protection, style, clarity, etc.) |
| What it is | 3–4 sentences, plain language |
| Who it's for | Vehicle types and use cases |
| Packages | Good / better / best from pricing guides |
| Process | Intake → install → delivery timeline |
| Warranty | Summary + link to full policy |
| CTA | Phone, website, book consultation |

**Source pricing tables from:** `ppf-pricing-guide`, `tint-pricing-guide`, `wrap-pricing-guide`, `coating-pricing-guide`, `detailing-pricing-guide`

## Social media templates

### Before / after post
- Image: split or swipe — before left, after right
- Caption: vehicle YMM, service performed, one customer benefit line
- Hashtags: 3–5 local + 3 service-specific
- CTA: "DM us for a quote on your [vehicle]"

### Education post
- Carousel: 3–5 slides explaining one concept (VLT, self-healing, hydrophobic, etc.)
- Pull facts from installation manual intro chapters

### Review spotlight
- Screenshot review (with permission)
- Thank customer by first name
- Tag shop location

**Posting cadence:** Minimum 3× per week; 1 education, 1 project, 1 trust/review.

## Vehicle photography standards

Per CS Manual Chapter 26 — minimum for every completed job:

<ul class="checklist">
<li>4 exterior angles in consistent lighting</li>
<li>2 detail shots (edges, finish, texture)</li>
<li>Before photos when applicable</li>
<li>No license plates visible unless customer consents</li>
<li>Photo release on file before marketing use (`photo-release-consent`)</li>
</ul>

## Referral program

Use **Referral Partner Agreement** for B2B (dealers, detailers, tint shops).

Customer referral script (CS Manual):
> "If you know anyone else who'd love this level of protection, send them our way — we take great care of referrals."

Track lead source in CRM: `Referral — Customer (Name)`.
