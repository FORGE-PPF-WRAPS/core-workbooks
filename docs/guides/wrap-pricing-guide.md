## Cost factors

| Factor | Impact |
|--------|--------|
| Film tier (cast vs calendared) | ±15–25% |
| Color / finish (matte, satin, chrome) | ±10–20% |
| Disassembly required | +$________ / hour |
| Design / print graphics | + artwork + print cost |
| PPF combo (wrap + PPF front) | Bundle discount 5–10% |
