# Reference documents

Drop your **existing completed SOPs and docs** here for conversion.

Supported formats for reference:
- `.md` — fastest to convert
- `.docx` / `.pdf` — agent will transcribe structure into markdown

When you send your document list, include:
1. Filename
2. Document title
3. Category (`sops`, `policies`, `checklists`, `forms`)
4. White-label only, or also needs branded variant

The agent will convert each file, register it in `manifest.json`, and push to the `cursor/sop-docs-d9e4` branch.
