# Chapter 166 — Technical Bulletin — Vendor material receiving


Mandatory shop bulletin for Vinyl Wrap Installation: **Vendor material receiving**. Effective immediately for all jobs.


## Purpose


Establish non-negotiable standards for vendor material receiving across all Vinyl Wrap Installation operations.


<div class="procedure">

### Vendor material receiving — Standard Workflow

1. Review bulletin before shift start
2. Verify tools and documentation available
3. Apply on every applicable job without exception
4. Log compliance on job card
5. Report deviations to lead installer same day

</div>


## Pass / Fail Criteria


| Checkpoint | Pass | Fail Action |
| --- | --- | --- |
| Documentation | Complete before keys returned | Hold delivery |
| Technique | Matches SOP | Rework before release |
| Safety | Zero violations | Stop work |


<div class="callout warning">

**Mandatory:** Vendor material receiving applies to retail, fleet, warranty rework, and mobile jobs without exception.

</div>
