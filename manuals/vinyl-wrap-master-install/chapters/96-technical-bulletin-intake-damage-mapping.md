# Chapter 96 — Technical Bulletin — Intake damage mapping


Mandatory shop bulletin for Vinyl Wrap Installation: **Intake damage mapping**. Effective immediately for all jobs.


## Purpose


Establish non-negotiable standards for intake damage mapping across all Vinyl Wrap Installation operations.


<div class="procedure">

### Intake damage mapping — Standard Workflow

1. Review bulletin before shift start
2. Verify tools and documentation available
3. Apply on every applicable job without exception
4. Log compliance on job card
5. Report deviations to lead installer same day

</div>


## Pass / Fail Criteria


| Checkpoint | Pass | Fail Action |
| --- | --- | --- |
| Documentation | Complete before keys returned | Hold delivery |
| Technique | Matches SOP | Rework before release |
| Safety | Zero violations | Stop work |


<div class="callout warning">

**Mandatory:** Intake damage mapping applies to retail, fleet, warranty rework, and mobile jobs without exception.

</div>
