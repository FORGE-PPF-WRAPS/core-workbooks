# Chapter 37 — Technical Bulletin 1 — Adhesion testing protocol


Shop technical bulletin for Vinyl Wrap Installation: **Adhesion testing protocol**. Distribute to all technicians. Post in bay until signed off.


## Purpose


Establish consistent shop standards for adhesion testing protocol across all Vinyl Wrap Installation jobs.


<div class="procedure">

### Adhesion testing protocol — Standard Workflow

1. Review bulletin with lead installer before shift
2. Verify all required tools and documentation are available
3. Execute procedure on next applicable job under observation
4. Log completion on job card or shop checklist
5. Report exceptions to shop manager same day
6. Suggest SOP improvement if friction identified

</div>


## Quality Standards


| Checkpoint | Pass Criteria | Fail Action |
| --- | --- | --- |
| Documentation | Complete before keys returned | Hold delivery |
| Technique | Matches shop SOP | Retrain before next job |
| Customer communication | Expectations set in writing | Manager call |
| Safety | Zero violations | Stop work immediately |


## Common Errors


<ul class="checklist">
<li>Skipping documentation "to save time"</li>
<li>Assuming previous job applies to current vehicle</li>
<li>Not escalating substrate concerns before starting</li>
<li>Using dull blades or expired chemicals</li>
<li>Delivering before QC sign-off</li>
</ul>


<div class="callout warning">

**Non-negotiable:** Adhesion testing protocol standards apply to every Vinyl Wrap Installation job — retail, fleet, and warranty rework.

</div>
