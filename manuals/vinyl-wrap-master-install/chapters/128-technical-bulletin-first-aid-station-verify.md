# Chapter 128 — Technical Bulletin — First aid station verify


Mandatory shop bulletin for Vinyl Wrap Installation: **First aid station verify**. Effective immediately for all jobs.


## Purpose


Establish non-negotiable standards for first aid station verify across all Vinyl Wrap Installation operations.


<div class="procedure">

### First aid station verify — Standard Workflow

1. Review bulletin before shift start
2. Verify tools and documentation available
3. Apply on every applicable job without exception
4. Log compliance on job card
5. Report deviations to lead installer same day

</div>


## Pass / Fail Criteria


| Checkpoint | Pass | Fail Action |
| --- | --- | --- |
| Documentation | Complete before keys returned | Hold delivery |
| Technique | Matches SOP | Rework before release |
| Safety | Zero violations | Stop work |


<div class="callout warning">

**Mandatory:** First aid station verify applies to retail, fleet, warranty rework, and mobile jobs without exception.

</div>
