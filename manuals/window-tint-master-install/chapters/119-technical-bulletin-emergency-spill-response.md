# Chapter 119 — Technical Bulletin — Emergency spill response


Mandatory shop bulletin for Window Tint Installation: **Emergency spill response**. Effective immediately for all jobs.


## Purpose


Establish non-negotiable standards for emergency spill response across all Window Tint Installation operations.


<div class="procedure">

### Emergency spill response — Standard Workflow

1. Review bulletin before shift start
2. Verify tools and documentation available
3. Apply on every applicable job without exception
4. Log compliance on job card
5. Report deviations to lead installer same day

</div>


## Pass / Fail Criteria


| Checkpoint | Pass | Fail Action |
| --- | --- | --- |
| Documentation | Complete before keys returned | Hold delivery |
| Technique | Matches SOP | Rework before release |
| Safety | Zero violations | Stop work |


<div class="callout warning">

**Mandatory:** Emergency spill response applies to retail, fleet, warranty rework, and mobile jobs without exception.

</div>
