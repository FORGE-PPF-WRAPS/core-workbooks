# Chapter 157 — Technical Bulletin — Estimator-to-floor handoff


Mandatory shop bulletin for Window Tint Installation: **Estimator-to-floor handoff**. Effective immediately for all jobs.


## Purpose


Establish non-negotiable standards for estimator-to-floor handoff across all Window Tint Installation operations.


<div class="procedure">

### Estimator-to-floor handoff — Standard Workflow

1. Review bulletin before shift start
2. Verify tools and documentation available
3. Apply on every applicable job without exception
4. Log compliance on job card
5. Report deviations to lead installer same day

</div>


## Pass / Fail Criteria


| Checkpoint | Pass | Fail Action |
| --- | --- | --- |
| Documentation | Complete before keys returned | Hold delivery |
| Technique | Matches SOP | Rework before release |
| Safety | Zero violations | Stop work |


<div class="callout warning">

**Mandatory:** Estimator-to-floor handoff applies to retail, fleet, warranty rework, and mobile jobs without exception.

</div>
