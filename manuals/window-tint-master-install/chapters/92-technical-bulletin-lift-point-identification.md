# Chapter 92 — Technical Bulletin — Lift point identification


Mandatory shop bulletin for Window Tint Installation: **Lift point identification**. Effective immediately for all jobs.


## Purpose


Establish non-negotiable standards for lift point identification across all Window Tint Installation operations.


<div class="procedure">

### Lift point identification — Standard Workflow

1. Review bulletin before shift start
2. Verify tools and documentation available
3. Apply on every applicable job without exception
4. Log compliance on job card
5. Report deviations to lead installer same day

</div>


## Pass / Fail Criteria


| Checkpoint | Pass | Fail Action |
| --- | --- | --- |
| Documentation | Complete before keys returned | Hold delivery |
| Technique | Matches SOP | Rework before release |
| Safety | Zero violations | Stop work |


<div class="callout warning">

**Mandatory:** Lift point identification applies to retail, fleet, warranty rework, and mobile jobs without exception.

</div>
