# Chapter 15 — Quality Control & Inspection Protocol


Cross-light method, edge audit, contamination scan, seam review, photo documentation, and sign-off checklist.


## Core Standards


<div class="procedure">

### Quality Control & Inspection Protocol — Shop Standard Sequence

1. Review work order and confirm vehicle/package match
2. Verify environmental conditions (temperature, humidity, cleanliness)
3. Stage all tools and materials before vehicle enters install zone
4. Execute procedure steps without skipping documented checkpoints
5. Document completion with photos where required
6. Lead installer signs QC before customer notification

</div>


## Reference Table — Key Decision Points


| Situation | Action | Escalate If |
| --- | --- | --- |
| Substrate questionable | Stop; test spot; disclose | Adhesion test fails |
| Customer adds panel mid-job | Change order signed before work | Refusal to sign |
| Contamination found mid-install | Stop; assess redo scope | Larger than dime zone |
| Pattern does not fit | Verify VIN/year; re-plot | Software confirms mismatch |
| Edge lift at QC | Re-tuck or section redo | Repeat on same panel type |

## Common Mistakes


### Troubleshooting Reference

| Symptom | Likely Cause | Corrective Action | Prevention |
| --- | --- | --- | --- |
| Chemical stain | Wrong cleaner on film | Replace area; educate | Provide aftercare sheet |
| Line at film seam | Poor butt-seam technique | Re-do seam | Follow seam standards |
| ADAS calibration fault | Bumper disturbed | Refer OEM calibration | Document sensor zones |
| Paint pull on removal | Weak factory repaint | Stop; heat; pro removal | Assess paint first |
