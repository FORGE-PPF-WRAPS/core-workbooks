# Chapter 92 — Technical Bulletin — Adhesion testing protocol


Mandatory shop bulletin for Customer Service: **Adhesion testing protocol**. Effective immediately for all jobs.


## Purpose


Establish non-negotiable standards for adhesion testing protocol across all Customer Service operations.


<div class="procedure">

### Adhesion testing protocol — Standard Workflow

1. Review bulletin before shift start
2. Verify tools and documentation available
3. Apply on every applicable job without exception
4. Log compliance on job card
5. Report deviations to lead installer same day

</div>


## Pass / Fail Criteria


| Checkpoint | Pass | Fail Action |
| --- | --- | --- |
| Documentation | Complete before keys returned | Hold delivery |
| Technique | Matches SOP | Rework before release |
| Safety | Zero violations | Stop work |


<div class="callout warning">

**Mandatory:** Adhesion testing protocol applies to retail, fleet, warranty rework, and mobile jobs without exception.

</div>
