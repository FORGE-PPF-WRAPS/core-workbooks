# Chapter 155 — Technical Bulletin — UV inspection protocol


Mandatory shop bulletin for Customer Service: **UV inspection protocol**. Effective immediately for all jobs.


## Purpose


Establish non-negotiable standards for uv inspection protocol across all Customer Service operations.


<div class="procedure">

### UV inspection protocol — Standard Workflow

1. Review bulletin before shift start
2. Verify tools and documentation available
3. Apply on every applicable job without exception
4. Log compliance on job card
5. Report deviations to lead installer same day

</div>


## Pass / Fail Criteria


| Checkpoint | Pass | Fail Action |
| --- | --- | --- |
| Documentation | Complete before keys returned | Hold delivery |
| Technique | Matches SOP | Rework before release |
| Safety | Zero violations | Stop work |


<div class="callout warning">

**Mandatory:** UV inspection protocol applies to retail, fleet, warranty rework, and mobile jobs without exception.

</div>
