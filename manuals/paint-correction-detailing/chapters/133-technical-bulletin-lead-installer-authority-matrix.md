# Chapter 133 — Technical Bulletin — Lead installer authority matrix


Mandatory shop bulletin for Paint Correction & Detailing: **Lead installer authority matrix**. Effective immediately for all jobs.


## Purpose


Establish non-negotiable standards for lead installer authority matrix across all Paint Correction & Detailing operations.


<div class="procedure">

### Lead installer authority matrix — Standard Workflow

1. Review bulletin before shift start
2. Verify tools and documentation available
3. Apply on every applicable job without exception
4. Log compliance on job card
5. Report deviations to lead installer same day

</div>


## Pass / Fail Criteria


| Checkpoint | Pass | Fail Action |
| --- | --- | --- |
| Documentation | Complete before keys returned | Hold delivery |
| Technique | Matches SOP | Rework before release |
| Safety | Zero violations | Stop work |


<div class="callout warning">

**Mandatory:** Lead installer authority matrix applies to retail, fleet, warranty rework, and mobile jobs without exception.

</div>
