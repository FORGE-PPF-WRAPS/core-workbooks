# Chapter 161 — Technical Bulletin — Comeback root cause coding


Mandatory shop bulletin for Ceramic Coating Application: **Comeback root cause coding**. Effective immediately for all jobs.


## Purpose


Establish non-negotiable standards for comeback root cause coding across all Ceramic Coating Application operations.


<div class="procedure">

### Comeback root cause coding — Standard Workflow

1. Review bulletin before shift start
2. Verify tools and documentation available
3. Apply on every applicable job without exception
4. Log compliance on job card
5. Report deviations to lead installer same day

</div>


## Pass / Fail Criteria


| Checkpoint | Pass | Fail Action |
| --- | --- | --- |
| Documentation | Complete before keys returned | Hold delivery |
| Technique | Matches SOP | Rework before release |
| Safety | Zero violations | Stop work |


<div class="callout warning">

**Mandatory:** Comeback root cause coding applies to retail, fleet, warranty rework, and mobile jobs without exception.

</div>
